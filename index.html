Questions

Do I need to pay for the AI features separately?
Yes — the AI runs on Anthropic's Claude API, which you access with your own API key. It's free to sign up and you pay only for what you use. Typical usage (scoring 10 leads, drafting 5 emails) costs a few cents. There's no monthly commitment.
Where is my data stored?
Entirely in your browser's localStorage — nothing goes to a server. Your contacts and deals never leave your device unless you use the AI features, which send only the relevant data to Anthropic's API (no personal identifiers beyond what you type).
Do I need to know how to code?
No. You open the file in a browser and it works. The PDF setup guide walks you through getting your Claude API key and optionally hosting on GitHub Pages (free), with no technical knowledge required.
What's the difference between this and HubSpot / Pipedrive?
Those are SaaS tools — you rent access monthly and your data lives on their servers. Meridian is a file you own. The AI is deeply integrated rather than a bolt-on, and there's no subscription. The trade-off: no email sync or calendar integration (yet).
Can I use this on my phone?
Yes. Meridian is fully responsive with a mobile-optimised layout and bottom navigation. Host it on GitHub Pages and access it from any device via browser.
Is there a refund policy?
Because this is a digital download with instant access, sales are final. If you have an issue, contact us via Gumroad and we'll do our best to help.
